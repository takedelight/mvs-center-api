# MVS Client API

API для системи **MVS Center**.

## Встановлення та запуск

### 1. Клонування та запуск API

```bash
git clone https://github.com/takedelight/mvs-center-api.git
cd mvs-center-api
npm install
```

Скопіюйте приклад конфігурації:

```bash
cp .env.example .env
```

Запустіть базу даних і сервер у Docker:

```bash
docker-compose up -d
```
